# Open Leo

*Open Leo* is a project for a digital library gathering Leonardo da Vinci's codices and manuscripts, for the Digital Scholarly Editing: Theory, Methods and Practice of the DHDK master’s degree at University of Bologna, A.Y. 2025/2026. 
Available at link [boloicoms.github.io/OpenLeo/](https://boloicoms.github.io/OpenLeo/).
